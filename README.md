# Photo Browser App

An exercise for displaying photos from [JSONPlaceholder API](https://jsonplaceholder.typicode.com/)

## Assignment requirements

* Create an SPA application for displaying photos
* There should be a possibility to share an URL that links to a single photo's page
* Recomended time used for this coding assingment is only couple of hours

---

## Tech Stack

- TypeScript
- REACT
- [React Router](https://reactrouter.com/)
- [Material UI](https://mui.com/material-ui/getting-started/)

## Installation

```bash
npm install
````

---

## Running the App

### Development mode

```bash
npm run dev
```

Server will start at:

```
http://localhost:5173
```

---

## Build & Run

```bash
npm run build
npm start
```

---

## Navigating in the application

### Main page
```
http://localhost:5173
```

### Single photo page
```
http://localhost:5173/photos/1
```
